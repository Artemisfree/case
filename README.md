## Описание
Приложение case собирает информацию со страниц в aliexpress.
После запуска приложения, отправляется интересующий запрос.
Приложение собирает основную информацию первых 10-ти ссылок запроса
и выдает в формате stdout

## Технологии в проекте 
* Python
* beautifulsoup4

## Запуск проекта
1. Клонировать репозиторий и перейти в него в командной строке
2. Установить зависимости из файла requirements.txt
3. Выполнить миграции
4. Запустить проект
